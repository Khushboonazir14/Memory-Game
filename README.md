# Memory Game

A simple and responsive Memory Card Game built using HTML, CSS, and JavaScript.

## Features

* Flip cards by clicking
* Match identical emoji cards
* Cards stay open after correct match
* Wrong cards flip back after 1 second
* Random shuffle on every game start
* Play Again button to restart the game
* Win alert when all cards are matched
* Responsive design for mobile devices

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

1. All cards start hidden with "?"
2. Click a card to reveal the emoji
3. Click another card to find its pair
4. If both cards match, they stay open
5. If they do not match, they flip back
6. Match all cards to win the game
